About This Detection System

This platform is built for real-world deepfake operations: media upload, model scoring, duplicate result reuse, persistence, and post-prediction feedback in one integrated workflow.

Prediction Pipeline

  • Audio feature extraction and model inference for .wav and .mp3 media.
  • Video frame analysis and model voting for .mp4 deepfake screening.
  • Fast duplicate detection using content hashing and cached response retrieval.

Deployment Focus

  • Google OAuth login for controlled access.
  • Mongo-backed prediction persistence with local fallback support.
  • Feedback loop for continuous data improvement and model iteration.
